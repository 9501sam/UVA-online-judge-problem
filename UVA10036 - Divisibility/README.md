# 題目意思<br>
給一長度為 n 的數列與一數 k，問在數列的相鄰兩數之間插入 '+' 或 '-'
<br>是否存在一運算法使得整個運算結果能夠被 k 整除。(1 <= n <= 10000, 2 <= k <= 100)
# 解題<br>
這題偏難，利用題目預設測資 
<br>2
<br>4 7
<br>17 5 -21 15
<br>4 5
<br>17 5 -21 15
<br>先看第一筆測資,m=4, k=7(除數), num=17,5,-21,15
<br>(17 \pm 5 \pm -21 \pm 15)%7可以看成 (17%7 + 5%7 + -21%7 + 15%7)%7
